# google-easy-register-and-login-web-page
In this page you can upload your embarassing moments like a social network while easily login with google account or your personal mail
You need postgresql setup and google authorization client key and secret set up in your .env file, it should look like this:

GOOGLE_CLIENT_ID="your client id"
GOOGLE_CLIENT_SECRET="your client secret"
SESSION_SECRET="your node.js/express session "
PG_USER="postgres"
PG_HOST="localhost"
PG_DATABASE="your database"
PG_PASSWORD="your password"
PG_PORT="your postgreport"



after you setup .env and sql infrastructure, from git bash terminal you can run "npm i" and "nodemon index.js" in order.
